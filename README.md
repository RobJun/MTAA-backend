### Config file

`config.json` must contain:

```json
{
    "database": {
        "host" : "<host>",
        "port" : "<port>",
        "name" : "<db name>",
        "user" : "<user>",
        "pass" : "<password>"
    },
    "hosts": [
        "<IPaddress>",
        ...
    ],

    "salt": "SECRET_KEY"
}
```
